Neovim Shortcut Reference
=========================

Current configured shortcuts.

Leader key: Space

Run / Format
------------
Ctrl+J       Toggle terminal
             If the run terminal already exists, toggle that terminal
             Otherwise open or close a normal shell terminal

F5           Run current file in the bottom terminal
             Python:      python <current file>
             Lua:         lua <current file>
             JavaScript:  node <current file>
             TypeScript:  tsx <current file>
             Go:          go run <current file>

F6           Format current file

Markdown Workflow
-----------------
These mappings are buffer-local and only exist in `filetype=markdown` buffers.

F5           Toggle live Markdown rendering in the current buffer
             Provided by `marklive.nvim`

<leader>mp   Toggle live Markdown rendering in the current buffer

Python Workflow
---------------
These mappings are buffer-local and only exist in `filetype=python` buffers.

F5           Run current Python file
             In visual mode: run selected code
F7           Run pytest on current file (requires pytest)
F8           Run pytest on current function (requires pytest)

<leader>pi   Organize imports through `ruff`
<leader>pr   Open Python REPL (interactive interpreter)

Go Workflow
-----------
These mappings are buffer-local and only exist in `filetype=go` buffers.

F7           Run `go test` in the current package directory
F8           Run the current `Test*`, `Benchmark*`, or `Example*` target

<leader>ga   Run `go test ./...` from the nearest module root
<leader>gr   Run `go run .` from the current package directory
<leader>gt   Switch between `foo.go` and `foo_test.go`
<leader>gi   Organize imports through `gopls`

LSP / Navigation
----------------
K            Show hover documentation
gd           Go to definition
gD           Go to declaration
gi           Go to implementation
gr           Show references

<leader>rn   Rename symbol
<leader>ca   Code action

[d           Previous diagnostic
]d           Next diagnostic
<leader>e    Show diagnostic for current line
<leader>q    Send diagnostics to location list

Completion
----------
<C-Space>    Trigger completion menu
<CR>         Confirm selected completion item
<Tab>        Next completion item / next snippet placeholder
<S-Tab>      Previous completion item / previous snippet placeholder
<C-e>        Cancel completion
